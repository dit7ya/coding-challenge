    Coolfold.foldr

    You're seeing just the function foldr, go back to Coolfold module for more information.
    Link to this function

    foldr(list, acc, f)

    Specs

    foldr([typeA], typeB, (typeA, typeB -> typeB)) :: typeB
when typeA: var, typeB: var

    The foldr function is like the foldl function but folds from the right instead. The results are different for non-associative functions. This implementation is not tail recursive and can cause stack overflows for very big input lists.

    Examples 

    iex> Coolfold.foldr([1,2,3,4], 0, fn x,y -> x+y end)
10

iex> Coolfold.foldr([1,2,3,4], 0, fn x,y -> x-y end)
-2
